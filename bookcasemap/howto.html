HowTo

Normal maps are based on OverPass Turbo data. The issue is not to overload that end point. So some caching is required. As a challenge I'm building it without a server, mainly using github.com for the hosting and Google Drive & Google Sheets for the data part.
  • Leaflet map, data in JSON files (structured?)
  • JSON files stored on a google drive
  • google script to update data, taken from OverPass Turbo
  • update a log of usage, filled from the map page
  • use the log to see what data needs to be updated
